About Me

Perkenalkan, nama saya Alif Akbar Hilmy Auliya. Saya adalah mahasiswa semester dua Universitas Negeri Yogyakarta prodi Pendidikan Teknik Informatika. Hobi saya sederhana yaitu menonton film dan bermain game. Walaupun hobi saya terkesan seperti orang pemalas tetapi saya rajin berolah raga, misalnya push up, pull up, dan jogging. Selain itu saya juga suka mempelajari pemrograman. Saat ini saya sedang fokus mempelajari pemrograman web dan kedepannya saya ingin mempelajari pemrograman aplikasi serta game.

Saya tertarik di bidang pemrograman karena pada saat bermain game saya pernah berpikir bagaimana sebuah visual dapa bergerak sesuai yang saya inginkan hanya dengan menekan tombol di layar hp. Selain itu saya juga sering melihat adegan film dimana seorang programmer atau hacker dapat mempermudah suatu hal hanya dengan menuliskan sebuah program. Saya sangat mengagumi karya yang dapat dibuat oleh para programmer seperti AI dan game. Oleh karena itu saya berkeinginan menjadi programmer profesional dan menmbuat game sesuai dengan imajinasi saya.
